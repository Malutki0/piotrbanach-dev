# 💜 Amorii

Strona e-commerce zrealizowana w pełni samodzielnie — od projektu graficznego po wdrożenie produkcyjne z integracjami płatności i wysyłki.

🔗 **Live:** [amoriiofficially.com](https://amoriiofficially.com)

---

## ✨ Co zostało zrobione

- 🎨 Projekt UI/UX w **Figmie**
- 💻 Kodowanie (HTML5, CSS3, JavaScript)
- ⚙️ Wdrożenie i konfiguracja **WordPress + WooCommerce**
- 💳 Integracja **Stripe API** — płatności online
- 📦 Integracja **InPost API** — generowanie paczek, śledzenie wysyłki
- 🔍 Optymalizacja **SEO** (meta tagi, struktura, sitemap)
- 🖼️ Obróbka grafiki i materiałów wizualnych
- 📱 Responsywny design (mobile-first)
